# Indian States Population Analysis

## Project Description
This project analyzes population distribution across Indian states and union territories.

## Files
- `notebooks/indian_population_analysis.ipynb` - Jupyter Notebook with analysis
- `visualizations/` - Contains generated charts
- `data/` - Dataset files

## Visualizations Created
1. Population distribution histogram
2. Gender distribution bar chart  
3. Sex ratio histogram

## How to Run
1. Open the Jupyter Notebook
2. Run all cells
3. View generated visualizations
